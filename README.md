
## Setup Backend

```cd social-publishing-api```

```cp .env.example .env```

```composer install```

```./vendor/bin/sail up```

```./vendor/bin/sail artisan migrate```


---

## Setup Frontend

Setup Frontend

```This project requires `npm 11.3.0```

```cd social-publishing```

```npm install```

```npm run dev```
